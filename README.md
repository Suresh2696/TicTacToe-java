Tic Tac Toe Game (Java)

This is a simple console-based Tic Tac Toe game developed using Java.

Description

Two players take turns entering row and column values to place X and O on the board.
The game checks for a win or draw after every move.

Features

Two-player game (X and O)

Runs in the console

Simple and easy logic

Beginner friendly

How to Use

Make sure Java is installed on your system

Open the project folder

Compile the program:

javac TicTacToe.java


Run the program:
